## Workflow Steps

### Step 1: Install MadHub
To begin, ensure you have MadHub installed on a real Android device, cloud phone, or emulator. This graphical interface tool requires no programming knowledge, allowing anyone to utilize its features efficiently.

### Step 2: Configure Instagram Auto-Reply
1. **Open MadHub**: Launch the MadHub application on your device.
2. **Select Instagram Auto-Reply**: Navigate to the Instagram Auto-Reply feature from the main dashboard.
3. **Set Configuration Parameters**:
   - **Input Mode**: Choose whether you will manually enter your reply messages or upload a file with predefined replies. This allows for flexibility based on your operational needs.
   - **Operation Interval**: Define how often the tool should check for new messages (e.g., every 5 minutes).
   - **Single Run Duration**: Specify the duration for which the auto-reply feature will be active (e.g., 2 hours).

### Step 3: Activate the Auto-Reply Feature
After configuring your parameters, simply activate the auto-reply feature. From this moment onward, MadHub will automatically detect and reply to unread messages according to your specifications.
